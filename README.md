# Installs [docker-compose 1.27.4](https://docs.docker.com/compose/) on [Ubuntu](https://www.ubuntu.com/)

```bash
wget -q https://script.install.devinsideyou.com/docker-compose
sudo chmod +x docker-compose && ./docker-compose
```
